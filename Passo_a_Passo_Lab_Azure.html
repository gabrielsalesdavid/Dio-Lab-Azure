1. Iniciando a Criação de Recurso no Azure

Ação realizada: Acesso ao portal Azure e clique no botão "Criar um recurso" ou "Create" para iniciar o processo de criação de uma nova máquina virtual.

2. Seleção de Máquina Virtual

Ação realizada: Busca e seleção da opção "Máquina Virtual" ou "Virtual Machine" no marketplace do Azure para prosseguir com a criação.

3. Configuração Básica - Parte 1

Ação realizada: Preenchimento das informações básicas como:
  • Assinatura (Subscription)
  • Grupo de Recursos (Resource Group)
  • Nome da Máquina Virtual
  • Região (Region)
  • Imagem do Sistema Operacional

4. Configuração Básica - Parte 2

Ação realizada: Continuação da configuração básica com:
  • Tamanho da Máquina Virtual (VM Size)
  • Tipo de Autenticação (SSH ou Senha)
  • Nome de usuário administrador
  • Opções de inicialização rápida

5. Configuração de Disco

Ação realizada: Configuração de armazenamento e disco:
  • Tipo de disco do SO (Disk type for OS)
  • Tamanho do disco
  • Criptografia de disco
  • Opções de backup e snapshot

6. Configuração de Rede - Parte 1

Ação realizada: Configuração inicial de conectividade de rede:
  • Seleção ou criação de Virtual Network (VNet)
  • Seleção de Subnet
  • Configuração de IP Público
  • Seleção de Grupo de Segurança de Rede (NSG)

7. Configuração de Rede - Parte 2

Ação realizada: Continuação das configurações de rede:
  • Definição de portas abertas (SSH, HTTP, HTTPS, etc.)
  • Regras de entrada (Inbound rules)
  • Verificação de conectividade
  • Balanceamento de carga (se necessário)

8. Configuração de Gerenciamento - Parte 1

Ação realizada: Configuração de ferramentas de gerenciamento:
  • Monitoramento (Azure Monitor)
  • Log Analytics Workspace
  • Azure Update Manager
  • Configuração de Diagnóstico

9. Configuração de Gerenciamento - Parte 2

Ação realizada: Continuação das configurações de gerenciamento:
  • Tags para organização de recursos
  • Políticas de conformidade
  • Alertas e notificações
  • Controle de Acesso baseado em Função (RBAC)

10. Configuração de NOC - Parte 1

Ação realizada: Configuração de Network Operations Center:
  • Análise de tráfego de rede
  • Monitoramento de performance
  • Visualização de topologia de rede
  • Troubleshooting de conectividade

11. Configuração de NOC - Parte 2

Ação realizada: Conclusão das configurações de NOC:
  • Análise de pacotes (Packet Capture)
  • Verificação de fluxo de tráfego
  • Diagnóstico de conectividade (Connection Troubleshoot)
  • Métricas de rede em tempo real

12. Criação de Chave SSH

Ação realizada: Geração e configuração de autenticação SSH:
  • Geração de par de chaves SSH (pública/privada)
  • Download da chave privada
  • Configuração de permissões de segurança
  • Armazenamento seguro da chave privada

13. Revisão de Configurações

Ação realizada: Revisão completa de todas as configurações:
  • Verificação de todas as configurações básicas
  • Revisão de disco, rede e gerenciamento
  • Confirmação de custos estimados
  • Validação de conformidade

14. Conclusão e Inicialização da VM

Ação realizada: Finalização da criação:
  • Clique no botão "Criar" ou "Create"
  • Aguardar o deployment/implantação
  • Confirmação de sucesso na criação
  • Recebimento de notificação de conclusão

15. Tela Inicial de Gerenciamento da VM

Ação realizada: Acesso ao dashboard de gerenciamento:
  • Visualização do painel da máquina virtual
  • Informações de IP público e privado
  • Status da máquina virtual (Running)
  • Acesso rápido a principais funcionalidades

16. Detalhes e Configurações da VM

Ação realizada: Exploração de detalhes da máquina:
  • Informações técnicas completas da VM
  • Configurações de armazenamento
  • Configurações de rede
  • Opções de gerenciamento e monitoramento

17. Iniciando Conexão SSH

Ação realizada: Preparação para conexão remota:
  • Obtenção do endereço IP público da VM
  • Preparação da chave SSH privada
  • Comando SSH para conexão: ssh -i [chave_privada] [usuário]@[IP_público]
  • Configuração do cliente SSH local

18. Máquina Virtual Conectada com Sucesso

Ação realizada: Conexão estabelecida com sucesso:
  • Conexão SSH estabelecida com a máquina virtual
  • Acesso ao terminal remoto
  • Possibilidade de executar comandos na VM
  • Máquina pronta para uso e configuração de aplicações
